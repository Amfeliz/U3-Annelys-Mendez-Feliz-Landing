# U3 Annelys Mendez Feliz/Landing
Sitio web, re-definición juego de mesa "Conectando" landing del Taller de Diseño de Interacción 2026

## Tecnologías Utilizadas

El proyecto está construido íntegramente con tecnologías web estándar y librerías de animación de alto rendimiento:

- **HTML5 & CSS3** (Diseño responsivo, CSS Grid, Flexbox, variables CSS nativas).
- **Vanilla JavaScript** (Lógica de interacción y manipulación del DOM).
- **[Three.js](https://threejs.org/)** (Renderizado WebGL para el entorno y las piezas 3D interactivas).
- **[GSAP](https://gsap.com/) (GreenSock Animation Platform)**:
  - *ScrollTrigger:* Para la coreografía animada atada al desplazamiento.
  - *TextPlugin:* Para el efecto fluido de escritura en los títulos.

## Instalación y Uso

Al ser un proyecto Front-end estático, no requiere un proceso complejo de instalación o dependencias por consola.

1. Clona este repositorio en tu máquina local:
   \`\`\`bash
   git clone https://github.com/tu-usuario/conectando-digital.git
   \`\`\`
2. Abre la carpeta del proyecto en tu editor de código preferido (ej. VS Code).
3. Para una visualización óptima (especialmente por las texturas e imágenes cargadas dinámicamente), se recomienda usar una extensión como **Live Server** o levantar un servidor local simple.
4. Explora la experiencia desde `index.html`.
